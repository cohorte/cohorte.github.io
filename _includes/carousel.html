Resilient Software Platform

COHORTE is a cross-platform runtime for executing and managing robust and reliable distributed Service-oriented Component-based applications. Faulty components are automatically isolated from other components to garantee continuty of service.

COHORTE uses iPOPO to develop Python-based Service-Oriented Component-based applications. The runtime infrastructure implements popular Java technology for the development of long-lived applications: OSGi Service Platform.
